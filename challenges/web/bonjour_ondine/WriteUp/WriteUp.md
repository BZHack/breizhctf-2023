### TL;DR

Le fichier robots.txt nous amène vers une page contenant une table HTML vide avec des cases ayant une class noire ou blanche. L'ajout d'attributs CSS sur ces classes permet de dévoiler un QRcode nous indiquant une 3eme page. Cette page contient un prompt en JS Fuck servant de mot de passe. Une fois décodé, l'utilisateur peut rentrer le mot de passe et récupérer le flag.

User-agent: *

L'accès à la page `ondine_secrets.html` nous emmène vers une page a priori vide:

![step2](step2.png)

L'inspection du document (ctrl+u) nous informe cependdant de la présence d'une table avec des classes `black` et `white`. 

![table](table.png)

L'ajout d'attributs CSS (dans l'inspecteur ou en sauvegardant une copie du document) nous permet d'ajouter une taille aux cellules ainsi qu'un fond de couleur correspondant à leurs classes:

```css
td{
    width: 10px;
    height: 10px;
}
.black{
    background: black;
}
```
